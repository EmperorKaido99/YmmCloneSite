# How to Add Your Videos

Drop your `.mp4` files into the `videos/` folder with these exact filenames:

| File | Used In |
|------|---------|
| `hero-background.mp4` | Hero section fullscreen background |
| `about-3dprinter.mp4` | About Us left column |
| `apply-drone.mp4` | Apply section right column |

## Tips
- Keep file sizes under 15MB each for fast loading (compress with HandBrake or CloudConvert).
- Resolution: 1280x720 or 1920x1080.
- Format: H.264 MP4 for maximum browser compatibility.
- Also add matching poster/fallback images to `images/`: `hero-poster.jpg`, `about-poster.jpg`, `apply-poster.jpg`.
