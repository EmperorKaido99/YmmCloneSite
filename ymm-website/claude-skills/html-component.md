# Skill: HTML Component

**Use when:** Adding a new section to `index.html`.

## Pattern
- Wrap section in `<section id="section-name">` with a semantic container `<div class="container">`.
- Use BEM-style class names: `.section__title`, `.section__body`, `.section__cta`.
- Keep markup lean — no unnecessary wrappers.
- Add a `<!-- SECTION NAME -->` comment above each section for readability.
