# Skill: CSS Responsive

**Use when:** Writing new CSS rules.

## Pattern
- Mobile-first: write base styles for small screens, then use `@media (min-width: Xpx)` to scale up.
- Use CSS custom properties (variables) defined in `:root` for colours and spacing.
- Breakpoints: 480px, 768px, 1024px, 1440px.
- Avoid fixed pixel heights on sections — prefer `min-height` with padding.
