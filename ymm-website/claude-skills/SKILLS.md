# SKILLS.md — YMM Claude Skills Index

Skills are reusable prompt templates for common development tasks on this project.

## How to Use
Tell Claude Code: "Use the skill in claude-skills/[file].md to [task]"

## Available Skills

| Skill File | Use When |
|---|---|
| `html-component.md` | Adding a new HTML section to index.html |
| `css-responsive.md` | Writing new CSS that matches existing breakpoints |
| `js-interaction.md` | Adding scroll animations or user interactions |
| `form-validation.md` | Adding or updating form validation logic |
| `video-section.md` | Adding a new video background to any section |
